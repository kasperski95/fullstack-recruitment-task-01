## Getting Started 
### Install Dependencies
- Node.js
- Redis Server

### Install Node Modules
```sh
client> npm i
server> npm i
```

### Run application
1. Start Redis Server
2. Start Apollo Server
    ```sh
    server> npm run dev # or npm start
    ```
3. Run App
    ```sh
    client> npm run dev
    ```
